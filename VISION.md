# fintrack — AI 소비 분석 가계부

## 프로젝트 방향

fintrack은 단순한 가계부 앱이 아니다.

기존 가계부처럼 숫자를 기록하는 서비스가 아니라,
AI가 사용자의 소비 흐름을 자동으로 정리하고 분석해주는
"AI 소비 분석 비서"를 목표로 한다.

---

## 핵심 철학

### 기존 가계부의 문제

- 직접 입력이 귀찮다
- 숫자만 많고 의미를 모르겠다
- 왜 돈이 안 모이는지 알기 어렵다
- 오래 사용하기 힘들다

---

## fintrack의 목표

사용자가 직접 관리하지 않아도:

- 거래내역 자동 정리
- 자동 카테고리 분류
- 실제 소비만 분석
- 소비 패턴 해석
- 맞춤 예산 설계
- 목표 기반 돈 관리

를 제공하는 것이 목표다.

---

## 핵심 컨셉

> **"AI가 내 소비 습관을 분석하고 나에게 맞는 돈 관리 방법을 설계해주는 가계부"**

---

## UX 핵심 원칙

### 1. 입력 최소화

사용자가 최대한 적게 행동해야 한다.

- 직접 입력 ❌
- 자동 정리 ⭕
- 자동 분류 ⭕
- 자동 분석 ⭕

### 2. 숫자보다 해석

단순 금액 표시보다 "왜 이런 소비가 발생했는지" 설명하는 것이 중요하다.

| ❌ | ⭕ |
|----|-----|
| 식비 420,000원 | 이번 달 식비 소비가 증가했어요. |
| | 늦은 시간 소비 비율이 높아요. |
| | 반복적인 배달 소비가 감지됐어요. |

### 3. 죄책감 유발 금지

앱의 말투는 사용자를 비난하거나 압박하면 안 된다.

| ❌ | ⭕ |
|----|-----|
| 과소비 중입니다. | 현재 소비 흐름 기준으로 조정 여지가 있어요. |
| | 이번 달은 변동 지출 비율이 높아요. |

### 4. "나를 이해해주는 느낌"

fintrack의 가장 중요한 감성.
단순 계산 앱이 아니라 "내 소비 패턴을 이해해주는 AI" 느낌이 중요하다.

## 핵심 기능 방향

### 1. 자동화 시스템

#### CSV 업로드 기반 자동 분석

사용자가 CSV만 업로드하면:

- 거래내역 분석
- 거래명 정리
- 자동 카테고리 분류
- 지출 제외 처리
- 반복 지출 탐지

가 자동으로 수행되어야 한다.

#### 거래명 자동 정리

- 체크카드 → 스타벅스 강남점
- 오픈뱅킹출금 → 카카오페이 송금

#### 자동 카테고리 분류

- 스타벅스 → 카페
- 배달의민족 → 식비
- 쿠팡 → 쇼핑
- 지하철 → 교통
- 넷플릭스 → 구독

룰 기반 + AI 혼합 구조로 설계한다.

#### 지출 제외 기능

실제 소비가 아닌 거래는 분석 대상에서 제외 가능해야 한다.

제외 대상 예시: 카드값 출금, 계좌이체, 중복 거래, 오픈뱅킹 출금

제외된 거래는:
- 목록에는 유지
- 통계/차트/AI 분석에서는 제외

#### 반복 거래 탐지

월세, 통신비, 보험, 넷플릭스 등 반복 거래를 자동 감지한다.

---

### 2. AI 소비 분석

fintrack의 핵심 기능.
단순 합계가 아니라 사용자의 소비 흐름을 자연어로 분석한다.

- 이번 달 식비 소비가 증가했어요.
- 월급 직후 소비 집중 패턴이 보여요.
- 새벽 시간 소비 비율이 높아요.
- 고정비 비율이 높은 편이에요.

---

### 3. 소비 성향 분석

사용자를 소비 패턴 기반으로 유형화한다.

- 감정 소비형
- 월급날 과소비형
- 소액 누수형
- 고정비 압박형
- 목표 달성형

---

### 4. 맞춤 재무 설계

현재 소비 패턴 기준으로 현실적인 예산을 추천한다.

> "현재 식비 평균은 68만원입니다. 다음 달 목표는 55만원부터 시작하는 것이 현실적입니다."

---

### 5. 목표 기반 시스템

사용자는 목표를 설정할 수 있어야 한다.

예시: 여행 자금, 비상금, 아이패드 구매, 독립 자금

AI는 현재 소비 흐름 기준으로:

- 예상 달성 기간
- 절약 가능 항목
- 목표 달성 확률

등을 분석한다.

---

### AI 금융 비서 방향

향후 사용자가 자연어로 질문 가능하도록 설계한다.

- 왜 돈이 안 모여?
- 어디 줄이면 돼?
- 내 소비 성향 알려줘.
- 다음 달 예산 짜줘.
- 이번 달 위험해?
